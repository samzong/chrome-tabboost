CSP and iframe compatibility settings

Modifies website response headers to remove Content Security Policy (CSP) and iframe restrictions, enabling link preview functionality.

This feature is enabled by default, allowing the extension to modify X-Frame-Options and Content-Security-Policy headers to enable pages to load properly in iframes.

WARNING
Note: Disabling this feature may cause link previews for some websites to fail, as those sites may prohibit iframe display. If you encounter preview loading issues, please enable this option.
